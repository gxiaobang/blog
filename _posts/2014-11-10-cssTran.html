在没有css3的transition，网页的绝大部分动画特效是用定时器，每隔一段时间改变element的css属性，还有一种就是flash。

结合css3的transition封闭的动画函数，用法如下 

	
	/**
	 * css3Tran(options);
	 * options配置属性
	 * 	elem		目标节点
	 * 	from		起始状态
	 * 	to  		目标状态
	 * 	duration	持续时间
	 * 	func		运动缓动函数
	 */
	css3Tran({
			elem: '#box',
			to: {
				width: 200
			}
		})
		.then({
			to: {
				transform: 'scale(2)',
				opacity: 0
			}
		})
		.complete(function() {
			new Dialog({log: '动画完成'});
		});
